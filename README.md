# IonClaw + DevDuo Platform

Integrated system: IonClaw Server + DevDuo AI + Orchestrator + Entanglement Twins.

## Structure
- `config/`: System constants.
- `ionclaw-server/`: Backend API.
- `devduo-ai-engine/`: Local AI runtime.
- `orchestrator/`: Central control.
- `entanglement-twins/`: State mirroring.
- `terminal-cli/`: CLI interface.
- `portal-ui/`: Web dashboard.
- `simulation-engine/`: Simulation scripts.
- `evaluation-suite/`: Testing tools.

## Quick Start
1. `npm install`
2. `npm run start-server`
3. `npm run start-cli`

## Simulation & Testing
- Run simulation: `node simulation-engine/simulator.js`
- Run tests: `node evaluation-suite/tester.js`
